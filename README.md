# Get Started

* Install python (Use whatever method you want. Make sure it is in system path).

* Install virtualenv
    ```
    pip install virtualenv
    \\ or
    pip3 install virtualenv
    ```

## MacOS

* Navigate to directory you wish to develop in.

mkdir
